# 野球スコア管理 PWA v2

複数デバイス間でリアルタイム同期できる野球スコア管理アプリです。

## 主な機能

### チーム・選手管理
- ✅ チーム登録・削除
- ✅ 選手登録・編集・削除
- ✅ 選手情報の五十音順表示
- ✅ 選手画像アップロード（1920x1080）
- ✅ 画像ギャラリー（スライド表示）
- ✅ 打撃・投手成績の一覧表示

### 試合管理
- ✅ 試合登録（日付・種別・場所・対戦相手）
- ✅ 公式戦の場合は大会名・回戦を記録
- ✅ 先攻・後攻選択
- ✅ 打順設定（最大9人）
- ✅ 試合削除

### 試合記録
- ✅ スコアボード（先攻・後攻対応）
- ✅ 10種類の打撃結果入力
- ✅ 打点・盗塁のカウンター入力
- ✅ 投手成績入力（投球回は0→1/3→2/3→1でカウント）
- ✅ 投手交代（複数投手の成績管理）
- ✅ 過去イニングの編集（スコアボードタップ）
- ✅ 打席結果の編集
- ✅ 試合終了・再開

### データ同期（Firebase）
- ✅ 複数デバイス間でリアルタイム同期
- ✅ オフライン対応（オンライン復帰時に自動同期）
- ✅ 画像もFirestoreに保存（Storage不要・無料プランで利用可能）

## セットアップ

### 1. GitHub Pages で公開

1. GitHubでリポジトリを作成
2. すべてのファイルをアップロード（フォルダではなくファイル単体で）
3. Settings → Pages → Branch: main → Save
4. 数分後にURLが発行される

### 2. Firebase同期を有効化（オプション）

`FIREBASE_SETUP.md` の手順に従ってFirebaseを設定すると、複数デバイス間でデータが同期されます。

Firebase設定なしでも、単一デバイスでは問題なく動作します（LocalStorage使用）。

### 3. アイコンを追加

以下のサイズのPNG画像を用意してください：
- `icon-192.png` (192x192px)
- `icon-512.png` (512x512px)

## 使い方

### ホーム画面への追加

**iPhone (Safari)**
1. Safariでサイトを開く
2. 共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」

**Android (Chrome)**
1. Chromeでサイトを開く
2. メニュー（︙）から「ホーム画面に追加」

### 基本的な流れ

1. **チーム作成** → チーム名を入力
2. **選手登録** → 名前・背番号・投手フラグ
3. **試合登録** → 日付・場所・対戦相手
4. **打順設定** → 選手をタップして追加
5. **投手選択** → 先発投手を選択
6. **試合開始** → 打撃結果を入力

### 試合中の操作

- **打撃入力**: ヒット、2塁打、HR などをタップ
- **打点・盗塁**: 下部のカウンターで入力後、打撃結果をタップ
- **チェンジ**: 攻守交代
- **投手交代**: 守備中に「投手交代」ボタン
- **イニング編集**: スコアボードのイニングをタップ
- **打席編集**: 打席結果をタップ
